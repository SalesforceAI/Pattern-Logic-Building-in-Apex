# Pattern & Logic Building in Apex

Pattern and logic building exercises in Apex using **for-loops**, **if-else**, and **switch** statements.  
This repository focuses on strengthening problem-solving skills through **star patterns**, **number series**, and **nested loop** examples.

## What’s Included
- Star pattern programs (triangle, inverted, pyramid, diamond)
- Number series / number patterns
- Nested loop practice
- Conditional variations using if-else
- Pattern selection using switch cases

## How to Run
1. Open Developer Console in Salesforce.
2. Create / paste the class from `src/classes`.
3. Run the method from Execute Anonymous, for example:
   - `StarPatterns.run();`
   - `NumberSeries.run();`
   - `SwitchPatternDemo.run(1);`
